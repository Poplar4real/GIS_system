# GIS_system
## 作者：Benjamin
## 功能：读取地图数据，绘制地图，放大，缩小，平移，在状态栏实时显示鼠标所在动态位置
## 坐标，控制鼠标放大缩小时的速率
## 开发工具：Microsoft Visual Studio 2017, Visual Basic .Net
